Detailed Description

template<typename T, typename S = T, typename U = T>
class Ignatus::Point< T, S, U >

Point is for using velocities and angle movement. Also used for coords.

This basicly as two values or a given type; x and y. This allows you to return two variables of the same type rather than only one value. Also has a few funtions for converting velocity to angles and speed. Supports basic math, like adding, also allows to check if Points have the same value.
